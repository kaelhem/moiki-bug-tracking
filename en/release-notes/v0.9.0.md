# v0.9.0

### [feat - editor]
+ You can now upload multiple files at once.  
+ In the media search, multiple items can now be selected simultaneously.  
+ Added the ability to record audio directly from the app.  
+ It’s now possible to add videos (search + upload).

### [fix - editor]
+ Fixed an issue that could cause a crash when deleting a routine while it was being displayed.  
+ Audio uploads with a “.webm” extension are now correctly recognized.

### [improvements - editor]
+ Review mode improvements:  
  - fixed display of sequence icons,  
  - added a search field,  
  - added a routine filter (shown when relevant),  
  - sequence blocks now include a direct link to their sequence.

### [improvements - player]
+ When switching themes, background images now feature a smooth cross-fade transition.  
+ Overall performance improvements.

### [fix - platform]
+ Fixed an issue preventing the import of stories published under Creative Commons
